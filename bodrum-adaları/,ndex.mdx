# Bodrum Adaları

```javascript
const fetchAdalar = async () => {
  try {
    const response = await fetch('https://api.airtable.com/v0/appSxejrZhNlWc0TL/Adalar', {
      headers: {
        'Authorization': `Bearer ${process.env.AIRTABLE_API_KEY}`,
        'Content-Type': 'application/json'
      }
    });

    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }

    const data = await response.json();
    return data.records;
  } catch (error) {
    console.error('Airtable API hatası:', error);
    return null;
  }
};

const adalar = fetchAdalar();
```

### Sedir Adası Turu
- Yalıçiftliği'nden Burgazada'ya
- Sedir Adası'na geçiş
- Gümüşlük Limanı'na dönüş

### Burgazada Turu
- Yalıkavak Limanı'ndan başlangıç
- Yalıçiftliği'ne geçiş
- Burgazada'ya uğrular
- Sedir Adası'na dönüş

### Yalıçiftliği Turu
- Bodrum Limanı'ndan başlangıç
- Yalıçiftliği'ne geçiş
- Burgazada'ya uğrular
- Sedir Adası'na dönüş
