## Günlük Tekne Turu Listesi

### Zorunlu
- [ ] Mayo (2 adet)
- [ ] Plaj havlusu (hızlı kuruyan)
- [ ] Güneş kremi (SPF 50+)
- [ ] Güneş gözlüğü
- [ ] Şapka (geniş kenarlı)

### Önerilen
- [ ] Deniz ayakkabısı
- [ ] Hafif hırka
- [ ] Su geçirmez telefon kılıfı
- [ ] Nakit
- [ ] İlaçlar (deniz tutması, ağrı kesici)

### Eğlence
- [ ] GoPro / su geçirmez kamera
- [ ] Şnorkel takımı
- [ ] Müzik çalar + kulaklık
- [ ] Kitap

## Mavi Yolculuk Listesi (3-7 Gün)

### Giysi
- [ ] Mayo (3-4 adet)
- [ ] Plaj havlusu (2 adet)
- [ ] Pareo / plaj elbisesi
- [ ] T-shirt (4-5 adet)
- [ ] Şort (3-4 adet)
- [ ] Elbise / etek (1-2)
- [ ] Pijama
- [ ] Hırka / ceket
- [ ] Spor ayakkabı + sandalet + terlik

### Kişisel Bakım
- [ ] Diş fırçası ve macun
- [ ] Şampuan + duş jeli
- [ ] Güneş kremi (SPF 50+, bol miktarda)
- [ ] After-sun losyonu
- [ ] Dudak koruyucu (SPF'li)
- [ ] Deodorant

### Sağlık
- [ ] Deniz tutması hapı
- [ ] Ağrı kesici + ateş düşürücü
- [ ] Antihistaminik
- [ ] Reçeteli ilaçlarınız
- [ ] Yara bandı + antiseptik krem
- [ ] Böcek kovucu

### Elektronik
- [ ] Telefon + şarj cihazı
- [ ] Powerbank
- [ ] Kulaklık
- [ ] Kamera + hafıza kartı
- [ ] Su geçirmez kılıf

### Belge
- [ ] Kimlik / pasaport
- [ ] Sağlık sigortası kartı
- [ ] Rezervasyon belgeleri

## Çocuklar İçin Ekstra

**0-6 Yaş:** Bebek bezi, ıslak mendil, mama, biberon, oyuncak, güneşlik şapka

**7-12 Yaş:** Yüzme kanatları, maske ve şnorkel, oyun kartları, tablet
